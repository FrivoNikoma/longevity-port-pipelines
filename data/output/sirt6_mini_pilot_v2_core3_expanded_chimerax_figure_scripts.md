# SIRT6 v2 core3-expanded ChimeraX figure scripts

## Purpose

This file indexes the ChimeraX figure scripts exported from the viewer-ready structure selections.

These scripts are intended to generate biologically interpretable 3D structure views in which:

- the target chain is shown as a light-gray cartoon;
- the partner chain is shown as a blue semi-transparent surface;
- the candidate residue patch is highlighted as orange-red spheres.

## How to use

Open a `.cxc` file in ChimeraX, or run it with:

```text
ChimeraX --script path/to/script.cxc
```

Each script also contains a `save` command to export a PNG rendering.
